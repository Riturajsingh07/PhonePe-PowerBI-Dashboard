# 📱 PhonePe Transaction Analysis Dashboard

## 📌 Project Overview

This project is an interactive Power BI dashboard developed to analyze PhonePe transaction data and generate meaningful business insights. The dashboard helps track transaction performance, user behavior, payment success rates, and service-wise transaction trends.

---

## 🎯 Objectives

- Monitor overall transaction performance
- Analyze transaction value trends over time
- Identify top-performing services
- Compare weekday vs weekend usage
- Understand age-segment contribution
- Track payment success and failure rates

---

## 📊 Key KPIs

- Total Transactions: 300K
- Total Transaction Value: ₹3.47 Billion
- Unique Users: 108K
- Success Rate: 96%

---

## 📈 Dashboard Features

✅ Monthly Transaction Trend Analysis

✅ Service-wise Transaction Value Analysis

✅ Age Segment Contribution Analysis

✅ Top 5 Users by Transaction Value

✅ Weekday vs Weekend Usage Analysis

✅ Payment Status Analysis

✅ Interactive Filters and Slicers

---

## 🛠️ Tools & Technologies

- Power BI
- DAX
- Data Modeling
- Data Visualization
- Excel / CSV Dataset

---

## 🔍 Key Insights

- Loans generate the highest transaction value.
- Millennials contribute the largest share of transactions.
- Weekday transactions are significantly higher than weekend transactions.
- Overall payment success rate remains above 96%.
